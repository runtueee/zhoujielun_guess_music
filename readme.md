# ? 周杰伦猜歌游戏 - Jay Chou Song Guessing Game

一个基于周杰伦歌曲的互动猜歌游戏，支持Web界面和命令行模式，专为周杰伦粉丝打造！

## ? 项目特色

- ? **双模式游戏**：选择题模式和填空题模式
- ? **三级难度**：简单(10s)、普通(5s)、地狱(2s)
- ? **积分系统**：答对获得积分，连胜有额外加成
- ? **连胜奖励**：10连胜和20连胜有特殊视频奖励
- ? **现代化界面**：基于Streamlit的美观Web界面
- ? **丰富曲库**：包含周杰伦多张经典专辑

## ? 快速开始

### 环境要求
- Python 3.7+
- FFmpeg（用于音频处理）

### 安装依赖
```bash
pip install streamlit pygame pydub mutagen
```

### 运行游戏

1. **Web版本**（推荐）：
```bash
streamlit run web_guess.py
```

2. **命令行版本**：
```bash
python guess_song.py.py
```


## ? 游戏玩法

### 开始游戏
1. 选择游戏模式（选择题/填空题）
2. 选择难度级别（简单/普通/地狱）
3. 点击"开始抽题"或自动开始

### 答题规则
- 系统会播放一段歌曲（根据难度2-10秒）
- 根据听到的片段猜测歌名
- 选择题模式：从4个选项中选择
- 填空题模式：直接输入歌名

### 积分系统
- 基础分：答对得10分
- 连胜加成：连续答对有额外加分
- 答错重置：答错后连胜次数清零

### 特殊奖励
- **10连胜**：解锁特殊庆祝视频
- **20连胜**：解锁更高级奖励视频

## ?? 技术实现

### 核心技术
- **Streamlit**：Web界面框架
- **Pygame**：音频播放
- **Pydub**：音频剪辑处理
- **Mutagen**：音频元数据读取
- **Base64**：音频数据编码

### 主要功能
- 智能音频剪辑：随机截取歌曲片段
- 动态难度调整：根据选择调整播放时长
- 状态管理：游戏进度和积分追踪
- 音效反馈：答对答错不同音效

## ? 自定义配置

### 添加新歌曲
1. 将MP3文件放入相应专辑文件夹
2. 确保文件名格式：`歌手 - 歌名.mp3`
3. 重启应用即可自动识别

### 修改游戏参数
在`web_guess.py`中可以修改：
- 积分计算规则
- 连胜奖励阈值
- 音频片段长度
- 界面样式和颜色

## ? 界面预览

### Web界面包含：
- **侧边栏**：战绩看板、难度选择、模式切换
- **主界面**：音频播放器、答题区域
- **反馈界面**：答题结果、积分变化、下一题按钮
